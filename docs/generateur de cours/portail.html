Quelle est la différence entre ces outils ?

Les Applications Principales (ci-dessus) sont des outils de CONCEPTION. Elles vous aident à créer des "prompts", c'est-à-dire des plans de construction détaillés pour vos cours ou vos jeux. Elles ne créent pas le fichier final mais vous donnent les instructions parfaites à utiliser avec une IA.

Le Créateur d'Exercices, par exemple, génère des prompts pour des exercices visuels à importer dans Léia/Néo, tandis que sa page secondaire, accessible via le bouton "Outils XML", permet de générer des prompts pour des formats plus techniques comme le XML Moodle.

Les Outils et Éditeurs (ci-dessous) sont des outils de PRODUCTION et de FINITION. Ils vous permettent de transformer les "prompts" en véritables fichiers HTML, d'éditer visuellement des pages existantes ou de simplement convertir un code en fichier téléchargeable.

Une plateforme en constante évolution

Ce portail est conçu pour être fonctionnel et flexible. Vous remarquerez peut-être que plusieurs chemins peuvent mener à des fonctionnalités similaires ; c'est voulu, pour vous permettre de choisir le parcours qui correspond le mieux à votre besoin du moment. L'ensemble de ces outils est en développement continu pour mieux vous assister.
